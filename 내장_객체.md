# global

노드에서 글로벌한 객체이다.

글로벌 객체는 생략할 수 있다.

~~~javascript
global.console.log('노드');
console.log('노드'); // 생략된 표현
~~~



# process

프로 세서 객체에는 여러 속성들이 있다. 

### proces.version

설치된 노드의 버전을 정보 

### process.arch

프로세스 아키텍처 정보

### process.platform

운영체제 정보

### process.pid

현재 프로세스의 아이디

### Process.uptime()

프로세스가 시작된 후 흐른 시간

### process.execPath

 노드의 풀 경로

### process.cwd()

현재 프로세스가 실행되는 위치

### process.cpuUsage()

현재 cpu 사용량
